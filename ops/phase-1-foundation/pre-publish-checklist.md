# Pre-Publish Checklist

Page path:
Owner:
Publish date:

## Content

- [ ] Title is unique and intent-aligned.
- [ ] Meta description is unique and clear.
- [ ] H1 is unique in page content.
- [ ] Body content is useful and non-duplicate.
- [ ] Local context is included where relevant.

## Linking

- [ ] At least 3 internal links included.
- [ ] Links point to relevant service/area/blog pages.
- [ ] Navigation and footer links remain valid.

## Schema

- [ ] JSON-LD exists for page type.
- [ ] Required fields are present.
- [ ] Values are consistent with visible page content.

## Conversion

- [ ] WhatsApp CTA exists and message ends with `(via pzm.ae)`.
- [ ] Call CTA exists where relevant.
- [ ] Contact section placeholder is present where expected.

## Technical

- [ ] Canonical is correct.
- [ ] Robots meta is intentional.
- [ ] OG/Twitter tags are complete.
- [ ] No broken media or obvious layout issues.

## Final Gate

- [ ] Publishing quality gate passed.
- [ ] Reviewer sign-off completed.
