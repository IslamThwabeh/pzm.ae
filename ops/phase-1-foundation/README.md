# Phase 1 - Foundation And Controls

This folder contains the operational assets to run Phase 1.

## Phase Goal

Create a consistent publishing and validation workflow before scaling content output.

## Included Assets

- `publishing-quality-gate.md`
- `pre-publish-checklist.md`
- `post-publish-validation.md`
- `roles-and-rituals.md`
- `weekly-kpi-report-template.md`
- `baseline-metrics-template.md`

## Definition Of Done For Phase 1

1. No page is published without passing the quality gate.
2. Baseline metrics are captured in week 1.
3. Weekly KPI review starts and runs every Friday.
4. Owners are assigned for content, QA, media, analytics, and publishing.
