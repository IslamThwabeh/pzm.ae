# Roles And Rituals

## Owners

- Content Lead: owns briefs and draft quality.
- SEO QA Lead: owns metadata, schema, and internal linking checks.
- Publisher: owns file updates, sitemap updates, and release packaging.
- Analytics Lead: owns GA4 and Search Console verification.
- Media Lead: owns real photos and asset hygiene.

## Weekly Rituals

### Monday - Planning

1. Select pages for this week.
2. Confirm target intents and primary keywords.
3. Assign owners and due dates.

### Tuesday/Wednesday - Production

1. Draft content and prepare media.
2. Complete schema and metadata.
3. Run pre-publish checklist.

### Thursday - Publish

1. Publish approved pages.
2. Update sitemap when needed.
3. Run post-publish validation.

### Friday - Review

1. Review KPIs and conversion events.
2. Record wins, issues, and next actions.
3. Prioritize next week backlog.

## Escalation Rules

1. Do not publish if quality gate fails.
2. Any schema inconsistency is a blocker.
3. Any broken CTA is a blocker.
